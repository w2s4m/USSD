## ⚡ البدء في 30 ثانية

### الطريقة 1: المتصفح المباشر
```
1. افتح المجلد
2. اضغط على index.html
3. استخدم التطبيق مباشرة
```

### الطريقة 2: خادم محلي
```bash
# الطريقة الأسهل - استخدم Python
cd USSD-Pay-v3
python -m http.server 8000

# ثم افتح: http://localhost:8000
```

### الطريقة 3: VS Code Live Server
```
1. اضغط كليك يمين على index.html
2. اختر "Open with Live Server"
3. سيفتح تلقائياً في المتصفح
```

## 📱 التثبيت على الجهاز

### على Android:
```
1. افتح في Chrome
2. القائمة (⋮) → تثبيت التطبيق
3. وافق
```

### على iPhone:
```
1. افتح في Safari
2. المشاركة → إضافة إلى الشاشة الرئيسية
3. اسم التطبيق → إضافة
```

## 🔧 استكشاف الأخطاء

### المشكلة: "يرفع الخادم صفحة فارغة"
```
الحل: تأكد أنك في مجلد USSD-Pay-v3 قبل تشغيل الخادم
```

### المشكلة: "لا يعمل Service Worker"
```
الحل: استخدم HTTPS أو localhost (الخادم المحلي)
```

### المشكلة: "الواجهة بطيئة"
```
الحل: امسح ذاكرة التخزين المؤقتة (Ctrl+Shift+Delete)
```

### المشكلة: "البيانات لم تُحفظ"
```
الحل: تأكد أن LocalStorage مفعل في إعدادات المتصفح
```

جوال باي تحتاج كلمة سر:
```
1. اطلب كلمة سر من جوال باي
2. عند التحويل الأول، ادخل الكلمة
3. تُحفظ محلياً في جهازك فقط
```
