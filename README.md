# USSD Pay v3.0

تطبيق ويب تفاعلي (PWA) لتحويل الأموال بسهولة عبر **جوال باي** و**بال باي**.

## ✨ المميزات

✅ **تطبيق PWA كامل**
- يعمل على المتصفح والجهاز
- يعمل بدون إنترنت (Offline-first)
- تثبيت تطبيق على الشاشة الرئيسية

✅ **واجهة عربية احترافية**
- RTL (يمين لليسار) كامل
- Dark/Light mode
- تصميم مستجيب (Responsive)
- حركات سلسة وانتقالات جميلة

✅ **إدارة سهلة للمستقبلين**
- حفظ وتنظيم المستقبلين
- المفضلات المميزة
- تصدير واستيراد النسخ الاحتياطية

✅ **سجل التحويلات**
- تتبع آخر 50 تحويل
- عرض/إخفاء السجل
- معلومات كاملة لكل عملية

✅ **أمان محلي 100%**
- جميع البيانات تُحفظ محلياً في الجهاز
- لا يتم إرسال أي بيانات للخوادم
- كلمة السر تُحفظ محلياً (اختياري)

## 🚀 البدء السريع

### 1. الاستخدام المباشر (بدون تثبيت)
```bash
# استنسخ المجلد أو حمّله مباشرة
# افتح index.html في المتصفح
```

### 2. الاستخدام عبر خادم محلي

**باستخدام Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**باستخدام Node.js:**
```bash
npx http-server -p 8000
```

**باستخدام Live Server (VS Code):**
- ثبّت extension "Live Server"
- اضغط يمين على `index.html` → "Open with Live Server"

ثم افتح المتصفح على: `http://localhost:8000`

## 📱 التثبيت على الجهاز

### Android/Chrome:
1. افتح التطبيق في Chrome
2. اضغط على القائمة (⋮)
3. اختر "تثبيت التطبيق" أو "إضافة إلى الشاشة الرئيسية"

### iOS/Safari:
1. افتح التطبيق في Safari
2. اضغط زر المشاركة
3. اختر "إضافة إلى الشاشة الرئيسية"
4. ادخل الاسم واضغط "إضافة"

## 🎮 طريقة الاستخدام

### الخطوة 1: اختر الخدمة
- جوال باي (Jawwal Pay)
- بال باي (Pal Pay)

### الخطوة 2: أدخل البيانات
- اسم المستقبل
- رقم الهاتف
- المبلغ (شيقل)

### الخطوة 3: احفظ أو حوّل
- **حفظ المستقبل**: للاستخدام اللاحق
- **تحويل الآن**: توليد كود USSD

### الخطوة 4: راجع الكود
- **انسخ**: انسخ الكود للصق في الهاتف
- **اتصال**: افتح الهاتف مباشرة
- **موافق**: أغلق النافذة

## ⚙️ الإعدادات

افتح **الإعدادات** (⚙️) لـ:

- 👥 **إدارة المستقبلين**: حذف أو جعل المفضلة
- 🔐 **الأمان**: حذف كلمة سر جوال باي
- 💾 **النسخة الاحتياطية**: تصدير/استيراد البيانات
- ℹ️ **معلومات التطبيق**
- 🔴 **إعادة التعيين**: حذف جميع البيانات (غير قابل للتراجع)
